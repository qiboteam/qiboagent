# building and testing

if you want to build and tes the rust core

```bash
cd core_with_backend/qibo_core
cargo build
cargo test
````

if you want to build the python package

```bash
cd core_with_backend/qibo_python
maturin develop
````

there are some python tests that simulate the exact circuit with the same syntax used in the original package.

## Building and Testing

### Prerequisites

* **Cargo** (Rust toolchain)
* **Python Interpreter**
    > **Note:** The maximum supported version for PyO3 is currently **3.12**.
* **Poetry**

### Build and test the Rust core with poetry

To build and test the Rust core, run:

```bash
cd core_with_backend/qibo_core
cargo build
cargo test
````

### Build the Python package with Poetry

First, install all dependencies:

```bash
poetry install
````
Compile the Python bindings:

```bash
poetry run maturin develop
```

Run the tests:
```bash
poetry run python -m pytest tests/test_bindings.py
```
